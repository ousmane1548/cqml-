Cette application permet de résoudre une équation du 2nd degré dans R. Il a été fait en Java Standard Edition avec une base de donnée embarquée HSQLDB. Ce document présente les différentes fonctionnalités de l'application.
1. Résolution : Permet de résoudre une équation du 2nd degré dans R.
Pour la résolution le coefficient a ne doit pas être nul. En cas d'erreur de saisie, une erreur vous sera notifier et on vous demandera de recommencer.
2. Historique : Permet de voir l'historique des résolutions.
3. Help : Affiche cet utilitaire.
